# Lesson 01 - Bounding Boxes

## Objective

Learn how to use bounding boxes for object detection.

---

## What I Learned

- A bounding box identifies the location of an object.
- A good bounding box fits tightly around the visible part of the object.
- Every object gets its own bounding box.
- Annotation guidelines always come first.

---

## Practice Exercises

### Exercise 1

Image:
- 🚗 Car
- 🚶 Person
- 🐶 Dog
- 🚗 Car

**Answer:**
I would draw **4 bounding boxes** because each object should have its own box.

---

### Exercise 2

Guideline:
> Annotate only cars.

**Answer:**
I would annotate only the cars and leave the other objects because I must follow the project instructions.

---

### Exercise 3

Scenario:
Half the car is hidden behind a wall.

**Answer:**
I would draw the bounding box around only the visible part of the car because that is the part I can accurately identify. Unless the project guidelines say otherwise, I should not guess the hidden portion.

---

## Reflection

Today I learned that image annotation is not just drawing rectangles. A high-quality bounding box must tightly fit the visible object and follow the annotation guidelines consistently.
